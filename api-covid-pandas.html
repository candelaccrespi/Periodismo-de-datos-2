Uso de la API del Covid19 con Pandas¶

API es la abreviatura para "Application Programming Interfaces" (su traduccin al espaol sera interfaz de programacin de aplicaciones). Se trata de un conjunto de definiciones y protocolos que se emplea para desarrollar e integrar el software de las aplicaciones. Esto nos permite la comunicacin entre dos aplicaciones de software por medio de un conjunto de reglas.

Pandas por su parte es la libera de Python especializada en el manejo y anlisis de estructuras de datos. Esta librera dispone de tres estructuras de datos diferentes:

  • Series: estrucutura de una dimensin.
  • DataFrame: Estructura de dos dimensiones (tablas.
  • Panel: Estructura de tres dimensiones (cubos).

En el presente trabajo se analizn datos extrados de Panda sobre el Covid-19

La URL: https://api.covid19api.com/countries
